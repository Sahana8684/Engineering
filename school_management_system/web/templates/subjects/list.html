Data Structures and Algorithms
CS201
4 Credits Core Subject

This course covers fundamental data structures such as arrays, linked lists, stacks, queues, trees, and graphs, along with algorithms for searching, sorting, and graph traversal. Students will learn algorithm analysis techniques and problem-solving strategies.

Learning Outcomes:
  • Implement and analyze common data structures
  • Apply appropriate data structures to solve computing problems
  • Analyze algorithm efficiency using Big O notation
  • Design efficient algorithms for various computational tasks
Faculty:

Dr. Sharma (Professor, CSE Department)

Database Management Systems
CS202
4 Credits Core Subject

This course introduces the principles of database design and management. Topics include data modeling, relational algebra, SQL, normalization, transaction management, and database security. Students will design and implement database applications.

Learning Outcomes:
  • Design and implement relational database schemas
  • Write complex SQL queries for data manipulation and retrieval
  • Apply normalization techniques to optimize database design
  • Implement transaction management and concurrency control
Faculty:

Prof. Verma (Associate Professor, CSE Department)

Object-Oriented Programming
CS203
3 Credits Core Subject

This course covers object-oriented programming concepts using Java. Topics include classes, objects, inheritance, polymorphism, encapsulation, interfaces, exception handling, and GUI programming. Students will develop applications using OOP principles.

Learning Outcomes:
  • Apply object-oriented design principles to software development
  • Implement class hierarchies using inheritance and interfaces
  • Develop applications with graphical user interfaces
  • Implement exception handling for robust applications
Faculty:

Dr. Gupta (Assistant Professor, CSE Department)

Computer Networks
CS204
4 Credits Core Subject

This course introduces the principles of computer networking. Topics include network architectures, protocols, TCP/IP model, routing algorithms, network security, and socket programming. Students will design and implement network applications.

Learning Outcomes:
  • Understand the layered architecture of computer networks
  • Analyze various networking protocols and their functions
  • Implement socket programming for client-server applications
  • Design and configure small to medium-sized networks
Faculty:

Dr. Patel (Professor, CSE Department)
