Syllabus Coverage

  • Introduction to Data Structures and Algorithm Analysis
  • Arrays and Linked Lists
  • Stacks and Queues
  • Trees (Binary Trees, Binary Search Trees, AVL Trees)
  • Graphs and Graph Algorithms
  • Sorting Algorithms (Bubble, Selection, Insertion, Merge, Quick, Heap)
  • Searching Algorithms (Linear, Binary, Hashing)
  • Algorithm Design Techniques (Divide and Conquer, Greedy, Dynamic Programming)

  • Introduction to Database Systems
  • Entity-Relationship Model
  • Relational Model and Relational Algebra
  • SQL (DDL, DML, DCL, TCL)
  • Database Design and Normalization (1NF, 2NF, 3NF, BCNF)
  • Transaction Management and Concurrency Control
  • Database Recovery
  • Indexing and Hashing

  • Introduction to Object-Oriented Programming
  • Classes, Objects, and Methods
  • Inheritance and Polymorphism
  • Abstraction and Encapsulation
  • Interfaces and Abstract Classes
  • Exception Handling
  • Multithreading
  • GUI Programming

  • Introduction to Computer Networks
  • Network Models (OSI and TCP/IP)
  • Physical Layer and Data Link Layer
  • Network Layer and Routing Algorithms
  • Transport Layer (TCP and UDP)
  • Application Layer Protocols
  • Network Security
  • Socket Programming
